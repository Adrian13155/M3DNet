* PyTorch == 1.2.0
* torchvision0.3.0
* Python3.6.0
* numpy1.19.5
* opencv-python3.3.0.10
* scikit-image0.14.4
* scipy1.2.1
* matplotlib3.1.0
* h5py3.1.0
